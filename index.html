Introduction

What is CSS?

CSS (Cascading Style Sheets) is code that's used to describe how HTML elements are displayed.

CSS isn't quite a programming language. It's more of a styling language used for web design, layout and variations in display according to the screen size.

How is CSS written?

The syntax for CSS is very simple and follows a basic pattern:

  1. Selector
  2. Declaration
    1. Property
    2. Value

The selector defines what HTML element should be styled, while the declaration block specifies what exactly about the HTML element should be styled, such as color or alignment:

p {color: red; text-align: center}

Colors

How can elements be colored?

There are several different methods of coloring elements, but the main three are the following:

  • Color name
  • Hex value
  • RGB value

There is a massive list of colors to choose from!

If you try copying the example code below, you will find that all 3 methods can be used to represent the same color!

Color name

Assigning color names to elements are the easist way to add color to them. However this method is limited due to the lack of customisation available.

Color names can only cover a small fraction of the massive variety that can be achieved by hex values and RGB values.

p {color: OrangeRed;}

Hex value

Hex values are 6 digit codes in hexadecimal format (from 0 to F) that specify to the computer what exact color to apply onto an element. This method is more advanced as it allows more precise color declarations. Hex values can allow up to 256 different colors.

The first 2 digits represent the red color, the second 2 digits represent the green color and the last 2 digits represent the blue color.

p {color: #FF4500;}

RGB value

RGB values allow 3 parameters in decimal format (from 0 to 255). This method is very similar to declaring hex values, but provides even more precision due to the range of values that can be given. RGB values can allow more than 16 million colors!

The first parameter represents the red color, the second parameter represents the green color and the last parameter represents the blue color.

p {color: rgb(255, 69, 0);}

Box Model

How are elements placed within boxes?

There are multiple boxes that surround an element with content. These different boxes can be manipulated to accomodate more space between elements or narrow down that space.

The main boxes in the order of their position relative to the elements are:

  1. Padding: Space between the element's contents and its border.
  2. Border: Line between the padding and margin of an element.
  3. Margin: Space between the element's border and surrounding elements.

All three boxes can be declared and manipulated for an element:

p {padding: 10px; border: 2px solid black; margin: 10px;}
